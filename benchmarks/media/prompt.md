# Media Benchmark Prompt

Build a modern digital publishing platform for media companies, newsrooms, and content creators. Features: article management (rich text, multimedia), editorial workflow, content scheduling, multi-author, categories/tags, SEO, paywall/subscriptions, newsletter, comments, social sharing, analytics, AMP, video/audio, podcasts, live blog, breaking news, content syndication, headless CMS, API, multi-site, personalization, A/B testing. Target: digital publishers, newsrooms, media companies.

## Key Requirements

### Content Management
- Article CRUD
- Rich text editor
- Media management
- Categories and tags

### Editorial Workflow
- Draft and review
- Approval process
- Scheduled publishing
- Version history

### Monetization
- Subscription plans
- Paywall enforcement
- Advertisement integration
- Newsletter management

### Engagement
- Comments system
- Social sharing
- Live blog
- Breaking news alerts

### Technical
- Headless CMS API
- Multi-site support
- A/B testing
- SEO optimization
