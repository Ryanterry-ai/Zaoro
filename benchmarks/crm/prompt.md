# CRM Benchmark Prompt

Build a modern sales CRM for B2B teams. Features: lead capture, pipeline management, contact management, deal tracking, email integration, activity logging, task management, reporting dashboards, forecasting, territory management, team collaboration, mobile app, and API for integrations. Target: sales teams at growing B2B companies.

## Key Requirements

### Core CRM
- Lead capture from web forms, email, imports
- Visual pipeline management with drag-and-drop
- Contact and account management with relationships
- Deal tracking with stages, probability, forecast
- Email integration (Gmail, Outlook) with tracking
- Activity logging (calls, meetings, emails, notes)
- Task management with reminders

### Analytics & Intelligence
- Reporting dashboards (pipeline, activity, revenue)
- Sales forecasting with weighted pipeline
- Territory management and assignment
- Team collaboration (mentions, sharing, teams)

### Technical
- Mobile responsive design
- REST API for integrations
- Webhook support
- Role-based access control
- Audit logging
- GDPR/CCPA compliance
- SOC 2 ready
