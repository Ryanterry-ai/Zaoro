# LMS Benchmark Prompt

Build a modern LMS for corporate training and course creators. Features: course builder (SCORM, xAPI, video, quizzes), learning paths, certifications, assessments, gamification, social learning, live sessions (Zoom/Teams), mobile app, reporting, e-commerce, multi-tenant, SSO, content library, authoring tool, compliance training, skills matrix. Target: corporate L&D and course creators.

## Key Requirements

### Course Builder
- Module/lesson structure
- Multiple content types
- Quiz and assessment builder
- SCORM/xAPI support

### Learning Paths
- Path creation
- Progression tracking
- Certification
- Completion certificates

### Engagement
- Discussion forums
- Social learning
- Gamification/badges
- Leaderboards

### Administration
- User management
- Enrollment management
- Progress tracking
- Reporting and analytics

### Platform
- Multi-tenant
- SSO integration
- White-label
- API access
