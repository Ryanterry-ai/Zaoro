# Education/LMS Benchmark Prompt

Build a modern LMS for online education. Features: course builder (video, text, quiz, assignment), student enrollment, progress tracking, certificates, discussions, live classes (Zoom/Teams), assessments, grading, analytics, corporate training, SCORM/xAPI, mobile app, white-label, marketplace, subscriptions, certificates, gamification. Target: course creators, training companies, universities.

## Key Requirements

### Course Management
- Course builder with modules/lessons
- Multiple content types (video, text, quiz)
- Assignment submission and grading
- Course certification
- SCORM/xAPI support

### Learning Experience
- Student enrollment
- Progress tracking
- Interactive quizzes
- Discussion forums
- Live sessions

### Monetization
- Course marketplace
- Subscription plans
- Coupons and discounts
- Affiliate program

### Analytics
- Student progress reports
- Course completion metrics
- Revenue analytics
- Assessment analytics
