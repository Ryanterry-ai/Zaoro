# Ecommerce Benchmark Prompt

Build a modern e-commerce platform for D2C brands. Features: product catalog (variants, bundles, subscriptions), cart/checkout (guest, express, saved payment), payments (Stripe, PayPal, Apple Pay, BNPL), orders (fulfillment, returns, exchanges), inventory (multi-warehouse, reservations), customers (accounts, wishlists, reviews, loyalty), marketing (coupons, abandoned cart, email/SMS, affiliates), analytics (revenue, conversion, LTV, cohort), admin dashboard, headless API, multi-currency, multi-language, SEO, accessibility. Target: $1M-$100M GMV brands.

## Key Requirements

### Core Commerce
- Product catalog with variants, bundles, subscriptions
- Shopping cart with guest and authenticated checkout
- Multiple payment methods (Stripe, PayPal, Apple Pay, BNPL)
- Order management with fulfillment, returns, exchanges
- Multi-warehouse inventory with reservations

### Customer Experience
- Customer accounts with order history
- Wishlists and product reviews
- Loyalty/rewards program
- Personalized recommendations

### Marketing & Growth
- Coupon and discount engine
- Abandoned cart recovery (email/SMS)
- Email and SMS marketing automation
- Affiliate/referral program

### Analytics & Operations
- Revenue, conversion, LTV, cohort analytics
- Admin dashboard for store management
- Headless API for custom frontends
- Multi-currency and multi-language support
- SEO optimization and accessibility compliance

## Technical Constraints
- TypeScript/React/Next.js stack
- PostgreSQL database
- Redis for caching/sessions
- Stripe for payments
- Responsive design (mobile-first)
- WCAG 2.1 AA accessibility
- GDPR/CCPA compliance
