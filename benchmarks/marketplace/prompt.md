# Marketplace Benchmark Prompt

Build a multi-vendor marketplace like Etsy/Airbnb. Features: vendor onboarding, product listings, search/discovery, cart/checkout, payments (split payouts), reviews/ratings, messaging, dispute resolution, analytics dashboard, subscription plans, commission management, shipping integration, mobile apps, admin moderation tools. Target: niche marketplace founders.

## Key Requirements

### Vendor Management
- Vendor application and approval flow
- Vendor dashboard with analytics
- Product listing management
- Payout management with commission

### Buyer Experience
- Search and discovery
- Product browsing with filters
- Shopping cart and checkout
- Order management
- Reviews and ratings
- Messaging to vendors

### Platform Operations
- Admin dashboard
- Vendor moderation
- Dispute resolution
- Commission management
- Analytics and reporting

### Technical
- Payment splitting (Stripe Connect)
- Shipping integration
- Mobile responsive
- API for integrations
- Multi-language support
