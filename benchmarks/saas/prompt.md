# SaaS Benchmark Prompt

Build a modern B2B SaaS platform for vertical software. Features: multi-tenant architecture, subscription billing, user management, RBAC, feature flags, usage metering, analytics, integrations marketplace, white-labeling, API, webhooks, admin dashboard, customer portal, onboarding flows, feature gating, usage-based pricing. Target: vertical SaaS founders.

## Key Requirements

### Multi-Tenancy
- Tenant isolation (data, config, branding)
- Tenant onboarding and management
- White-label per tenant
- Subdomains and custom domains

### Billing & Subscriptions
- Plan management (tiered, usage-based)
- Subscription lifecycle (trial, active, paused, cancelled)
- Invoice generation
- Payment processing (Stripe/Paddle)
- Usage metering and tracking

### User Management
- User invitation and authentication
- Role-based access control (RBAC)
- Team management per tenant
- SSO integration (SAML, OAuth)

### Developer Platform
- API key management
- Webhook configuration
- Documentation
- Rate limiting

### Analytics
- Usage analytics per tenant
- Feature adoption metrics
- Revenue analytics
- Custom reports
