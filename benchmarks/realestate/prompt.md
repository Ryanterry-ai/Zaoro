# Real Estate Benchmark Prompt

Build a modern real estate platform for agents, brokers, and property managers. Features: property listings (MLS/IDX integration), search with maps, virtual tours, lead capture, CRM, transaction management, document management, e-signature, commission tracking, marketing automation, website builder, mobile app, reporting. Target: independent agents to mid-size brokerages.

## Key Requirements

### Property Listings
- MLS/IDX integration
- Property CRUD management
- Photo/video/virtual tour media
- Property search with maps
- Advanced filters

### Lead Management
- Lead capture forms
- CRM pipeline
- Lead routing and assignment
- Automated lead distribution
- Lead scoring

### Transaction Management
- Transaction pipeline
- Document management
- E-signature integration
- Task checklists
- Timeline tracking

### Marketing
- Email campaigns
- Social media integration
- Website builder
- Property flyers
- Lead nurturing sequences

### Reporting
- Sales analytics
- Agent performance
- Lead conversion metrics
- Commission tracking
