# Fintech Benchmark Prompt

Build a modern fintech platform for digital banking. Features: user onboarding/KYC, accounts (checking, savings, crypto), cards (virtual/physical), payments (ACH, wire, RTP, crypto), transfers, budgeting, savings goals, investing, lending, rewards, business accounts, API platform, webhooks, compliance dashboard. Target: neobank/fintech startups.

## Key Requirements

### Account Management
- Multi-account types (checking, savings)
- Crypto wallet support
- Account statements
- Transaction history

### Payments
- ACH/wire transfers
- RTP instant payments
- Crypto transactions
- Card management

### Financial Tools
- Budgeting and tracking
- Savings goals
- Investment platform
- Lending products
- Rewards program

### Compliance & Security
- KYC/AML onboarding
- Fraud detection
- Audit logging
- Regulatory compliance

### Technical
- API platform
- Webhooks
- Real-time notifications
- SOC 2 ready
